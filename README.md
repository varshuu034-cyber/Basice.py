# 💰 Expense Tracker (CLI)

A simple command-line based Expense Tracker built in Python. This program allows users to add expenses, view all recorded expenses, and calculate the total amount spent.

## 📌 Features

- ➕ Add new expenses  
- 📋 View all expenses  
- 💵 View total expenses  
- ❌ Exit the application
